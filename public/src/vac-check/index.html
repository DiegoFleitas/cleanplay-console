Flyio VAC Check

A simple way to check if players in a TF2 (os CS:GO) match have any VACbans.

F.A.Q

How to use this site?

  • While in game, open the console and run the status command.
  • Copy the output, we only really need the user data like so: # 809 "❣г๏zเt๏❣" [U:1:172149372] 29:11 100 0 active. Avoid the headers of the list like # userid name uniqueid connected ping loss.
  • Paste the list into the prompt and click accept.
  • That's it! the list shows if any player was banned recently among other stuff ;).

Why do I need this?

I wanted an easy way to check for cheaters, since finding them became something frequent on public servers.
